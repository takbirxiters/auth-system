Auth By Takbir - Preferred UI + Netlify GitHub backend

Upload this folder/ZIP to Netlify.

Set Environment Variables in Netlify:
GITHUB_TOKEN = your fine-grained GitHub token
GITHUB_OWNER = takbirxiters
GITHUB_REPO = USER-AND-PASS
GITHUB_FILE = credentials
GITHUB_BRANCH = main

Token permission:
Fine-grained GitHub token with Contents: Read and Write for only this repo.
